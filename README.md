# Seller Insights AI 

An AI-powered analytics dashboard for e-commerce sellers. Built in 2 days to demonstrate product thinking, full-stack development, and commerce understanding for Square's APM program.

## 🎯 Problem

Small business sellers have transaction data, but they don't know what to do with it. They see "28% cancellation rate" and don't know if that's good or bad—or what action to take.

## 💡 Solution

A dashboard that analyzes seller metrics and generates actionable insights.

## 🏗️ Architecture

- Frontend: React 18 + Tailwind CSS (dark theme)
- Backend: Flask (Python) REST API
- Data: In-memory JSON (10 sample sellers)

## 🚀 Features

- Seller selection dropdown (10 different business types)
- 4 key metric cards (completion rate, revenue, repeat customer rate, cancellation rate)
- AI-style insights with benchmarks and recommendations
- Priority action highlighting
- Professional dark theme UI

## 📊 Sample Sellers

10 realistic sellers across different industries:
- Coffee shop, boutique, repair shop, yoga studio, bakery
- Marketing agency, vinyl records, salon, furniture, gym

Each with realistic metrics and AI-generated insights.

## 🛠️ Tech Stack

Frontend: React, Tailwind CSS, Axios
Backend: Flask, Python
Data: In-memory JSON (no database)
